Welcome to the **Lovelace** project documentation.

## Project Structure

Lovelace is a Turborepo monorepo hosting multiple React apps, a shared UI library, and an API.

- **Admin**: Internal management interface.
- **Hub**: Main user interface.
- **Playtest**: Dedicated playtest platform.
- **API**: Backend services.
- **UI Library**: Shared components built with Radix UI and Tailwind v4.

## Tech Stack

- **Framework**: React + Vite (Apps) / Astro + Starlight (Docs)
- **Styling**: Tailwind CSS v4
- **Package Manager**: Bun
- **Monorepo Tooling**: Turborepo
